# MATHeng

A mathematical tool that can solve equations, find derivatives, find integrals, and find extrema.

## Features

- Solve equations
- Find derivatives
- Find integrals
- Find extrema
- Graph functions

## Installation

1. Clone the repository
2. Install the required packages
3. Run the application

## Usage

1. Run the application
2. Select a tool from the menu
3. Enter the function
4. Solve the problem
